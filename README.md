# E-Commerce Analytics Dashboard

## Project Overview
This project analyzes e-commerce sales, customer orders, funnel performance, and monthly targets using Power BI and CSV datasets.

## Tools Used
- Power BI
- CSV Data
- Data Cleaning
- Business Analytics
- Dashboard Design

## Files Included
- ecommerce_orders.csv
- ecommerce_funnel_daily.csv
- ecommerce_monthly_targets.csv
- Power BI dashboard file

## Dashboard Features
- Sales performance tracking
- Funnel conversion analysis
- Monthly target comparison
- Business performance insights

## Folder Structure
- data: Contains CSV dataset files
- dashboard: Contains Power BI dashboard file
- images: Contains dashboard screenshots
- docs: Contains extra project documentation

## Key Learnings
- Built an interactive dashboard in Power BI
- Analyzed e-commerce sales and customer funnel data
- Compared monthly targets with actual performance
- Improved business reporting and visualization skills
